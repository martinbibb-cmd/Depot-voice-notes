Depot schema summary

Sections

Edit the list of depot note sections as JSON. Normally this is an array of section names: ["Needs", "Working at heights", ...]. "Future plans" will always be added as the last section automatically.

Checklist

Edit the checklist configuration as JSON. This should be an object like: { "sectionsOrder": [...], "items": [...] }. Items are the same shape used by the app (id, label, section, group, hint, etc).
